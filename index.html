DPSS Stress Detection

Real-Time Emotion Recognition

This project implements a real-time stress detection system combining audio and video analysis to evaluate emotional states. The audio module utilizes librosa for feature extraction and a 1D CNN to classify stress levels from speech, while the video module uses grayscale images and a 2D CNN for facial emotion recognition. The system is designed for applications in dispatcher mental health monitoring, offering insights into stress levels and mood states based on real-time input. Project Link

Magic Seeds

Smart Plant Care Platform (Co-Developed)

Magic Seeds is a smart plant care web platform designed to help plant lovers manage their plants with ease. Users can add plants, set custom watering schedules, and receive reminders through the MagicLoops API framework for text notifications. The platform includes dynamic popups for managing plant details, a profile popup for account management, and randomized plant images to keep the interface fresh. Although full SMS integration wasn't implemented due to resource limitations, the project establishes a solid foundation for future updates. Project Link.
